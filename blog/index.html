Red Gate is the sixth planned upgrade, and the fourth online upgrade of Kulupu blockchain.

Runtime change#

  • Enable voluntary taxation. The taxation is initially set to 0% and can be increase through a democracy referendum.
  • Set technical committee to be directly appointed by democracy module.
  • Set council treasury approval threshold to be 50%, and reject threshold to be 20%.
